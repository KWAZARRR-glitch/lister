# 📝 List Creator — Создание пользовательских списков

Простое и удобное приложение с графическим интерфейсом для создания списков любой длины с возможностью выбора формата вывода результатов.

## ✨ Возможности

- ✅ Создание списков с любым количеством элементов (до 100+)
- 🎨 **4 формата вывода** на выбор:
  - 📋 Столбик с номерами
  - 🐍 Python-список (`['a', 'b', 'c']`)
  - ➡️ Строка с разделителем (`a → b → c`)
  - 📊 Таблица (по 5 элементов в строке)
- 🔄 Прокрутка для больших списков
- 🎯 Интуитивно понятный интерфейс
- 🚀 Работает как скрипт или как standalone `.exe`
